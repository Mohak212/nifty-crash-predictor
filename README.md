# Nifty 50 Market Crash Predictor

AI-powered system to predict stock market crash probability using market data and news sentiment.

## Features
- ML model using Nifty 50 + India VIX data
- NLP sentiment analysis on financial news
- Backtesting engine (Sharpe ratio, drawdown)
- Streamlit dashboard for visualization

## Tech Stack
Python, XGBoost, NLP, Streamlit
